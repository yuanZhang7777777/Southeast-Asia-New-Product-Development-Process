## 6. MVP 建议范围

第一阶段业务验收只做前置决策闭环：

```text
待认领数据(入池)
  → 分配 / 销售自领
  → 销售认领
  → 主管审核
  → 产出“可备货清单”（对齐《海外仓备货申请表》，先导 Excel；含开品邮件状态标记）
```

> 校正（2026-06-29）：MVP 前段终点是 **“出可备货清单”**；**备货 / 采购 / 供应链 / 下单 = 外部桥**（平台只导出 + 登记状态），到货刊登属后段。后续链路按 `docs/08-端到端流程链路规划.md`（两段 + 一桥）对齐，不作为第一阶段验收。

第一阶段提醒只做：

- 分配后提醒销售认领。
- 销售提交后提醒主管审核。
- 所有逾期任务提醒责任人并升级主管。

备货、到货、缺货、广告费、1-4 周监控和四周总结后置。
